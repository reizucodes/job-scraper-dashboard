# Frontend (Vue 3 + TypeScript + Tailwind)

## Local setup
1. Install dependencies: `npm install`.

## Run
- Dev server: `npm run dev`
- Tests: `npm run test`
- Typecheck/build: `npm run build`

## Environment
- `VITE_API_BASE_URL` (optional): backend base URL, defaults to `http://localhost:8000`.
